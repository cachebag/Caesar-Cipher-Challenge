# Caesar Cipher System 

Quest 3 in Wayne State University's CSC 1100 course. 

## How to Run

### Windows
1. Compile:
   ```bash
   g++ -o caesar caesar_cipher_system.cpp
   ```
2. Run:
   ```bash
   caesar.exe
   ```

### macOS/Linux
1. Compile:
   ```bash
   g++ -o caesar caesar_cipher_system.cpp
   ```
2. Run:
   ```bash
   ./caesar
   ```
